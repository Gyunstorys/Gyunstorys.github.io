1. Code

1.1. Java 외부프로세스 실행 InputStream to OutputStream

개요: Springboot로 web server를 하나 두어 분석하는 프로그램을 개발 중에 있는 중 자바에서 분석기를 실행하였을 때 System.out 으로 출력되는 Log를 HDFS에 적재하려고 함.

과정:

  • HDFS JAVA API는 createFile() method를 실행 시 OutputStream 객체를 Return
  • ProcessBuilder의 stream에 outputstream을 넣을 수 있는 method가 있는 지 확인 하였으나 File 객체만 삽입 가능 하였음.
  • 참고문서 1번의 안전한 방법 중 apache common exec library의 예시를 찾아 봄.
  • Executor의 stream handler를 통하여 InputStream과 OutputStream을 핸들링 가능 한것을 확인하였고 참고문서 2번에서 PumpStreamHandler를 생성 하여 사용한것을 확인
  • PumpStreamHandler의 설명은 참고문서 3번을 확인하면 sub-process의 표준 입출력을 상위 프로세스로 복사하는 데 사용된다고 나와있다.

코드 : 

   public void run() {
        try {
            this.executor = new DefaultExecutor();
            CommandLine cmdLine = CommandLine.parse(arguments);
            PumpStreamHandler psh = new PumpStreamHandler(os, os);
            executor.setStreamHandler(psh);
            executor.execute(cmdLine);
        }catch (Exception e){
            /**
             * TODO: 실패 시 처리 필요.
             */
            e.printStackTrace();
        }finally {
            IOUtils.closeQuietly(os);
        }
    }
